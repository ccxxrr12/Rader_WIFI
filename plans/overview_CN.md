## 技术栈

### 后端技术
- **框架**: FastAPI (Python 3.9+)
- **神经网络**: PyTorch 2.0+, TensorFlow 2.x
- **数据库**: PostgreSQL 14+, Redis(缓存)
- **消息队列**: RabbitMQ/Celery
- **CSI处理**: NumPy, SciPy, 自定义C++模块
- **测试**: pytest, pytest-asyncio, pytest-mock

### 前端技术
- **框架**: React 18+ with TypeScript
- **状态管理**: Redux Toolkit
- **UI组件**: Material-UI v5
- **实时**: Socket.IO客户端
- **测试**: Jest, React Testing Library, Cypress

### 基础设施
- **容器化**: Docker, Docker Compose
- **编排**: Kubernetes
- **CI/CD**: GitHub Actions
- **监控**: Prometheus, Grafana
- **日志**: ELK Stack (Elasticsearch, Logstash, Kibana)
