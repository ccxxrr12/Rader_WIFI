# NAME of the project

## Description

本项目为一个基于卡内基梅隆大学于2022年发表的DensePose From WiFi的端侧适配项目，以WIFI CSI数据作为感知输入，对目标建筑物wifi信号覆盖范围内的碳基生物，主要以人体为目标对象进行感知，通过分析人体运动引起的CSI扰动,我们可利用基于物理的信号处理和机器学习实时重建身体位置、呼吸频率、心率和存在状态。

## Background

### 国内外研究现状
#### 一、近五年核心技术研究进展

本领域近五年的核心技术突破围绕 “理论突破 - 精度提升 - 端侧适配 - 场景落地” 的全链条展开，国内外研究团队在不同细分方向形成了各自优势，共同推动技术体系的完善与成熟。

##### 1.1 基础理论体系持续完善，原创性突破支撑技术落地

近五年，全球学界完成了 WiFi 感知核心理论体系的迭代升级，从早期的经验性模型向精准化、可量化的理论体系跨越，为技术落地奠定了坚实的理论根基。

- **国际层面**，MIT、斯坦福大学、加州大学伯克利分校等团队在 2021-2024 年完善了 WiFi 信号传播与人体感知的映射模型，明确了菲涅尔区感知边界、多普勒频移与人体微运动的量化关系、多径效应下有效信号的分离准则，解决了传统模型感知精度不可控、场景适配性差的核心问题；2025 年，佐治亚理工学院团队进一步完善了多节点协同感知的时空同步理论，为全屋多设备分布式感知提供了理论支撑。

- **国内层面**，北京大学张大庆教授团队在 2022 年提出了完整的 WiFi 菲涅尔区感知理论体系，相关成果发表于国际顶会 MobiCom 并获最佳论文奖第二名，是国内团队在该领域的首次顶级会议奖项突破，该理论精准揭示了 WiFi 信号感知的极限与核心影响因素，大幅提升了复杂场景下的感知稳定性；2024 年，该团队进一步提出 WiFi BFM/BFI 无线感知新理论模型，突破了商用 WiFi 设备的感知数据采集限制，无需硬件修改即可实现更高精度的信号采集，相关成果发表于 UbiComp 2024，实现了基础理论的原创性引领。

##### 1.2 人体行为与姿态感知从粗分类向精细化重建跨越

近五年，该方向完成了从 “简单动作分类” 到 “精细化人体姿态重建” 的技术跨越，识别场景从视距环境拓展到穿墙、遮挡等真实家居非视距环境，是全球研究最集中的细分领域。

- **精细化感知方面**，2022 年卡内基梅隆大学（CMU）团队在 CVPR 发表里程碑式成果《DensePose From WiFi》，首次基于商用 WiFi 设备的 CSI 数据实现人体 24 个关键区域的 3D 姿态重建，精度接近低分辨率摄像头方案，相关开源项目截至 2026 年累计获超 3.7 万星标；2023-2025 年，斯坦福大学、清华大学团队先后优化了非视距场景下的姿态估计算法，实现了穿墙场景下的多人动作轨迹追踪与姿态重建，解决了传统方案视距受限的核心痛点。

- **落地场景适配方面**，2022-2025 年，加州大学洛杉矶分校、西北工业大学团队先后针对居家核心监护需求，优化了跌倒检测算法，实验室环境下跌倒检测准确率均突破 98%，误报率低于 1%；2025 年，天津大学团队开发了基于商用 WiFi 设备的全屋人体轨迹追踪算法，将遮挡环境下的室内定位误差降低了 42%，适配真实家庭的复杂户型结构；同期，西北工业大学团队提出的多状态感知系统，在真实家居混合场景下实现了 97.57% 的人群计数准确率，多人行为识别准确率超 93%，突破了传统方案单人监测的局限。

##### 1.3 生理体征监测技术突破理想环境限制，向真实家居场景深度适配

针对呼吸、心跳带来的亚分贝级超微弱 WiFi 信号提取难题，近五年全球研究突破了实验室理想环境限制，重点攻克了非视距、多干扰、多目标场景下的稳定监测难题，拓展了技术在居家健康管理领域的应用边界。

- **核心算法突破方面**，2023 年 MIT CSAIL 实验室团队提出多子载波融合自适应滤波算法，首次实现穿墙 3 个房间、非视距场景下的人体呼吸与心率稳定监测，监测误差分别控制在 ±1 次 / 分钟、±2 次 / 分钟以内；2022-2024 年，清华大学、电子科技大学团队先后开发了小波变换与深度学习融合的微弱信号提取算法、盲源分离抗干扰算法，在真实家居卧室、客厅等非视距场景下，实现了呼吸频率监测误差≤±1.5 次 / 分钟、心率监测误差≤±3 次 / 分钟的稳定性能；2025 年，西安电子科技大学团队开发的自适应滤波算法，将动态家电干扰环境下的体征监测稳定度提升了 38%。

- **能力边界拓展方面**，2024 年华盛顿大学团队实现了同一场景下 3 人的同步呼吸、心率分离监测，单人监测准确率超 92%；2025 年佐治亚理工学院团队进一步突破，实现了基于 WiFi 信号的血压、血氧饱和度等进阶生理指标的无创监测，将 WiFi 感知从基础生命体征监测拓展到全维度居家健康管理。

##### 1.4 嵌入式端侧轻量化技术成熟，实现低资源设备端侧实时推理

针对商用智能家居设备嵌入式芯片资源受限的核心问题，近五年全球研究完成了从云端推理到端侧实时推理的技术跨越，为 WiFi 感知能力原生集成到现有智能家居设备奠定了核心基础，也是本项目核心方向的重要技术支撑。

- **模型轻量化方面**，2022 年加州大学洛杉矶分校（UCLA）团队提出基于模型剪枝与 INT8 量化的轻量化多任务感知模型，将模型体积压缩至 4.8MB，首次在 ESP32-S3 嵌入式 WiFi 模块上实现行为识别、跌倒检测的端侧实时推理，单帧推理延迟低于 80ms；2024 年，电子科技大学团队针对国产瑞芯微 RK 系列嵌入式芯片、OpenWrt 开源路由系统，开发了全链路轻量化感知算法框架，在资源受限的家用路由器上实现了行为识别、跌倒检测、生理监测三大任务的端侧实时推理，单帧推理延迟低于 100ms，可直接部署于市面主流国产智能家居设备。

- **硬件级适配方面**，2023-2024 年，高通、英特尔在最新的 WiFi 7 商用芯片中集成了感知加速引擎，实现了 CSI 数据采集、预处理、AI 推理的全链路芯片级加速；2025 年，德克萨斯大学奥斯汀分校团队开发了面向 OpenWrt 系统的通用感知算法适配框架，可直接在 90% 以上的商用家用路由器上部署，无需硬件改造，大幅降低了技术落地门槛。

##### 1.5 复杂场景泛化性与鲁棒性攻坚，破解规模化落地核心瓶颈

针对实验室模型在真实家庭场景泛化能力差、部署门槛高的行业痛点，近五年全球研究重点聚焦零标定、零样本泛化、抗环境干扰等核心方向，为技术规模化普及扫清障碍。

- **跨场景泛化方面**，2025 年北京大学团队开发了跨环境零样本学习算法，无需重新采集数据训练，即可在全新家居场景下实现 94% 以上的动作识别准确率，解决了传统模型跨房间、跨户型精度骤降的核心问题；2023-2024 年，加州大学圣塔芭芭拉分校团队提出的域自适应算法，将动态环境变化下的模型精度损失降低了 60% 以上。

- **低门槛部署方面**，2023-2025 年，天津大学团队先后开发了零标定感知算法、基于扫地机器人的环境测绘与信号地图构建方法，无需专业人员上门标定，普通用户即可完成系统自主部署，彻底解决了传统方案部署门槛高的痛点；2024 年，华为中央研究院团队发布了商用 WiFi 设备的即插即用感知适配方案，无需修改设备固件即可实现基础感知能力。

#### 二、近五年标准化与产业化落地进展

##### 2.1 全球标准化体系正式建成，国内配套标准同步推进

标准化是技术规模化落地的核心前提，近五年全球与国内的标准体系同步建设，形成了 “国际统一标准 + 国内场景化配套” 的格局。

- **国际层面**，WiFi 联盟于 2021 年正式启动 IEEE 802.11bf 标准制定工作，2025 年 9 月正式发布全球首个 WiFi 感知国际标准 IEEE Std 802.11bf-2025。该标准定义了 WiFi Sensing 的底层通信协议、数据格式、设备交互规范、安全与隐私保护框架，为全球商用 WiFi 设备原生支持感知能力提供了统一技术标准，彻底解决了此前不同厂商设备碎片化、不兼容的核心问题。

- **国内层面**，2022-2025 年，中国通信标准化协会（CCSA）先后启动多项 WiFi 感知相关行业标准立项，聚焦智能家居、智慧养老场景的技术规范、数据安全与隐私保护要求，与 IEEE 802.11bf 国际标准形成互补，推动技术的国产化落地与场景适配。

##### 2.2 产业化布局全面铺开，消费级场景落地加速

近五年，全球头部科技企业已完成核心技术布局，从芯片级适配到终端产品落地形成了完整的产业链，技术从实验室逐步走进民用消费场景。

- **国际市场**，2022-2025 年，亚马逊、谷歌先后在其 Echo、Nest 系列智能音箱、智能家居中控设备中集成 WiFi 感知功能，面向北美市场推出独居老人跌倒检测、居家安防告警商用服务，累计落地超 200 万家庭；2025 年，加拿大初创企业 Cognitive Systems 推出的 WiFi 全屋感知产品，已与北美前三大运营商达成合作，累计部署超 150 万个家庭节点；高通、英特尔的 WiFi 7 感知加速芯片已向全球路由器厂商开放商用授权。

- **国内市场**，2023-2026 年，华为、小米、中兴等头部企业先后在其 WiFi 6/7 旗舰路由器中预留了 WiFi 感知硬件与软件接口，完成了人体存在检测、跌倒检测、睡眠监测等功能的原型开发，部分高端智能家居套装已试点集成 WiFi 感知联动功能；2024-2025 年，北京、上海、天津、西安等多地民政部门与社区养老服务中心，已试点基于 WiFi 感知的独居老人监护方案，在超 200 个社区完成落地，实现了老人跌倒实时告警、居家活动状态监测等功能，取得了良好的社会效果。
## Meaning && Purpose

当前主流家庭安防方案均存在显著的场景局限，无法适配全屋智能家居的安防需求，本系统可针对性解决核心痛点，具备极强的应用必要性与适配性：

- **接触式边界安防设备的场景盲区问题**：门窗磁、红外对射、机械锁等传统设备仅能实现出入口的边界防护，无法覆盖室内全空间，无法识别已突破边界的非法入侵人员；仅能触发开关量报警，无法还原入侵人员的位置、移动轨迹，且易被规避 —— 入侵者可通过非出入口破拆进入室内，或触发报警后静止躲藏，直接导致系统失效。本系统通过全屋 WiFi 信号的全域覆盖，可实现室内全空间的人体探测，无论入侵者通过何种途径进入室内、是否保持静止，均可精准识别，彻底解决边界防护的盲区问题。

- **视频监控方案的隐私与环境适配局限**：摄像头是当前家庭安防的主流方案，但存在不可逆的隐私泄露风险，完全无法适配卧室、卫生间等私密空间的安防需求，易被物理遮挡、断电断网规避，且夜间、逆光环境下识别精度大幅下降；同时依赖用户主动查看，无法实现全时段主动预警，误报率高，无法精准区分人体与其他移动物体。本系统仅解析 WiFi 信号的 CSI 特征数据，不采集任何图像、音频信息，从根源上杜绝隐私泄露风险，可适配全屋所有私密空间，不受光线、遮挡、环境变化影响，实现 24 小时不间断的无感监测。

- **专用无源探测设备的成本与部署门槛问题**：PIR 红外传感器仅能识别运动人体，无法检测静止、躲藏的入侵者，极易被规避；毫米波雷达虽可实现高精度探测，但需要额外部署专用硬件，单设备覆盖范围有限，全屋部署成本高，且需要布线、专业调试，无法与现有智能家居系统深度融合。本系统直接复用家庭已有的智能家居设备的 WiFi 芯片与硬件资源，无需额外部署专用传感器，无需二次布线与调试，大幅降低安防系统的部署成本与门槛，可无缝接入现有智能家居生态，实现安防与家居控制的深度联动。

- **现有 WiFi 安防方案的复用性与覆盖能力不足**：当前已有的 WiFi 感知安防方案，大多仅依赖单一无线路由器作为收发节点，覆盖范围有限，多径效应干扰严重，识别精度不足，无法适配大户型、多房间的全屋场景，且未实现与全屋智能家居设备的深度复用，无法发挥多节点协同的感知优势。本系统将全屋所有具备 CSI 感知能力的 WiFi 智能家居设备转化为分布式感知节点，通过多节点协同探测，大幅提升覆盖范围与识别精度，完美适配不同户型的家庭场景，真正实现全屋无死角安防。

#### 本系统相较于传统方案的核心应用优势：

- 极致的硬件复用性，部署成本与门槛极低
- 全场景无死角覆盖，安防可靠性大幅提升
- 原生隐私友好设计，适配全屋私密空间
- 无感式监测体验，不干扰正常家居生活
- 深度兼容现有智能家居生态，可扩展性极强

## Usage
本研究设计的基于 WiFi 信道状态信息（CSI）感知的全屋智能安全防盗系统，核心应用定位为：面向现代家庭全屋智能家居场景，深度复用家庭已规模化部署的、具备 CSI 感知能力的 WiFi 芯片的智能家居设备（包括无线路由器、智能网关、智能灯具、智能开关、大家电智能终端等），无需额外部署专用安防硬件，通过多节点 WiFi 信号的 CSI 特征扰动解析，实现无源、非接触、全场景覆盖的人体存在、行为轨迹与异常动作识别，构建一套低成本、高隐私性、无感知、无监测死角的家庭智能安全防盗体系，为家庭住宅提供全时段、全场景的入侵预警、异常行为识别与安防联动服务，填补传统家庭防盗方案在智能家居场景下的应用短板。
### 核心落地应用场景：
- 非法入侵实时检测与多等级预警
- 无人值守状态下的全屋智能安防布控
- 异常滞留与高危破坏行为的智能识别报警
- 紧急胁迫场景的隐蔽式无声报警
- 多场景自适应布防与智能家居联动
